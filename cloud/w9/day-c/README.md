# W9 Day C - Progressive Delivery and Canary

Study date: Wednesday, 2026-06-10
Track: Self-study D3 Progressive Delivery

This folder packages the Day C work for W9. The goal is to understand how to release safely with canary deployment, metric-based analysis, and automatic abort behavior.

## Folder contents
- `plan-2026-06-10.md`: time-boxed study and implementation plan
- `tasks.md`: required and optional checklist
- `evidence.md`: record of notes, outputs, and follow-up questions
- `rollout/README.md`: notes for Argo Rollouts and Rollout resources
- `analysis-template/README.md`: notes for metric analysis and abort criteria
- `notes/README.md`: summary notes and mapping back to W8

1. Explain progressive delivery and canary release in simple language.
2. Explain the role of Argo Rollouts and the Rollout CRD.

4. Draft one AnalysisTemplate using Prometheus-style metrics.
5. Explain how bad metrics can trigger automatic abort.
